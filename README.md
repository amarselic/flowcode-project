## 🧠 About Project

This is a simple stopwatch (stoperica) made in Flowcode.  
It counts time in seconds and milliseconds and can be started, stopped, and reset.

---

## ⚙️ Features

- ▶️ Start stopwatch
- ⏸️ Pause/Stop time
- 🔄 Reset counter
- ⏱️ Accurate time counting
- 🧩 Built using Flowcode logic

---

## 🛠️ Technologies Used

- Flowcode
- Microcontroller simulation (if used)
- Basic logic circuits

---

## ▶️ How to Run

1. Download the repository
2. Open project in Flowcode
3. Run simulation
4. Use Start / Stop / Reset buttons
